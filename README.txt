Description:
function init():initializes all the variables and calls the function render and setupShape
function setupShape():stores the octahedron vertices and the colors for each vertice in their own matrix
function rotateShapeX(), function rotateShapeY(), and function rotateShapeZ():these functions use the XYZ keys which rotates the octahedron on the x axis, y axis, and z axis for the duration that it is pressed
function translateShape():function to make shape move  using the WASD keys 
function scaleShape():function to make shape scale using the I key to scale Y and O key to scale X 
function Movekeys():function that sets the keys to move the shape  
function render():call this function repeatedly for animation
